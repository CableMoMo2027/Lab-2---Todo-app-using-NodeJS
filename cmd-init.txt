=== body-parser ===
What is body-parser?
- Body-parser is middleware that parses incoming request bodies in a middleware chain.
- It processes data sent in HTTP requests, such as form data or JSON.

Why use body-parser?
- To extract data from POST requests (e.g., forms).
- To parse JSON or URL-encoded payloads.

=== EJS (Embedded JavaScript) ===
What is EJS?
- EJS is a simple templating engine for generating HTML with embedded JavaScript.
- It allows you to dynamically generate HTML based on server-side data.

Why use EJS?
- To build dynamic web pages.
- To pass data from the server to the client in HTML format.
